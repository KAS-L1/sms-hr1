# native-php-framework

A framework that can build web application faster using native php advance 🚀

### 💻 Tech Stack

-   HTML, CSS, JS
-   Bootstrap
-   JQuery Ajax
-   PHP (OOP)
-   MySQL Database
-   Laragon Web Server
